## 2024-05-23 - [TUI Navigation Affordances]
**Learning:** In Text User Interfaces (TUI), users lack standard visual cues like buttons or clickable links. Modal states (like search or dialogs) must explicitly list available keyboard actions (e.g., "[Esc] Cancel", "[Enter] Confirm") directly adjacent to the active element to prevent user anxiety and "trap" feelings.
**Action:** Always pair interactive TUI inputs or modal states with a persistent helper text line listing primary navigation keys.
